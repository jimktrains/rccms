Troubleshooting

If you find you're having problems getting DataMapper to work for you, be sure to check that your server meets the Server Requirements and that you have correctly followed the Installation Instructions.

If you have not yet read through all the topics in the General Topics section of the Table of Contents, I recommend you do this now. There might be some small gem of information you haven't seen which will solve your issue. I'd also suggest comparing your code against the examples to ensure you are using DataMapper as it was designed.

Have you seen this error?

If you're getting an error that looks very similar to the one below, it's because you need to update your CodeIgniter MySQL Driver with the latest version from the CodeIgniter SVN.

A Database Error Occurred

Error Number: 1054

Unknown column 'table.*' in 'field list'

How to update your MySQL Driver:

  1. Download the latest version of the mysql_driver.php file from the CodeIgniter SVN.
  2. Upload it to your CodeIgniter system/database/drivers/mysql folder.

That's it! The error should now be gone.
