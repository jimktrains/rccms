Road Map

The full functionality I wanted for DataMapper is already done but as with most things, there's always room for improvement!

Here's a list of tasks I have planned for future versions of DataMapper:

  • An automated, easy way of formatting data for HTML.
  • A series of related query clauses, for example: related_where(), related_like() etc.
  • More comprehensive examples covering validation, plus a real world example site.
  • Automatic Database table creation via class variable $database.

If you have any suggestions or ideas on what you'd like to see in DataMapper, feel free to contact me!
