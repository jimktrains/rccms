Installation Instructions

Installing DX Auth library with all example

  1. Unzip the package.
  2. Copy captcha folder into your CI folder.
  3. Copy system folder into your CI system folder.
  4. Install DX Auth database schema into your database.
  5. Open the application/config/config.php. Change $config['sess_use_database'] to TRUE.

Installing only DX Auth library

  1. Unzip the package.
  2. Copy captcha folder into your CI folder.
  3. Copy system/plugins/ folder into your CI system/plugins/ folder.
  4. Copy system/application/config/ folder into your CI application/config/ folder.
  5. Copy system/application/libraries/ folder into your CI application/libraries/ folder.
  6. Copy system/application/helpers/ folder into your CI application/helpers/ folder.
  7. Copy system/application/models/ folder into your CI application/models/ folder.
  8. Copy system/application/language/ folder into your CI application/language/ folder.
  9. Install DX Auth database schema into your database.
  10. Open the application/config/config.php. Change $config['sess_use_database'] to TRUE.

That's it!

If you're new to DX Auth, please read the Getting Started section of the User Guide to begin using DX Auth.
